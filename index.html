STAGE 01

40ft 集装箱定位与桩基点交

40ft HC 集装箱落位于 150mm 工业加厚混凝土地坪上，精准对齐 50x75ft 建筑网格。

STAGE 02

波纹板等离子切割与 100mm 方钢补强

等离子切割开出 36'×16' 大厅与屏风窗框，切口周圈焊接 100mm×100mm RHS 方钢防止变形。

STAGE 03

H-Beam 柱网与 900sqft Mezzanine 阁楼

立 150mm×150mm H-beam 钢柱，在后区 35ft 高顶钢构区（檐高 24ft）铺设 Bondek 钢承板搭建二层 Loft 办公平台（一楼/二楼净高各 11ft）。

STAGE 04

30° 高坡屋顶与 Raised Jack Roof 拔风塔

架设轻钢坡屋顶与中央拔风塔，敷设 50mm PU 聚氨酯防热隔音夹芯板。

STAGE 05

马六甲 Batu Angin 通风花砖屏风墙砌筑

在切割开口框内砌筑赤陶与白色通风花砖，形成防热透风双层外墙。

STAGE 06

热带仓储别墅完工交付

施工粗抹灰暖沙漆与热带庭院，实现自然降温 3-5°C 节能度假体验。

STYLE PACKAGE 02 (RECOMMENDED MAIN FOCUS)

风格 2：马六甲度假村风砖屏风墙 (Breeze Block Screen Wall)

材质组合：橄榄森林绿 PU 隔热屋顶 + 赤陶与白色 Batu Angin 通风花砖屏风墙 + 暖沙色粗抹灰质感漆

1. 外观正立面 (Front Exterior Elevation)

森林绿屋顶与红陶/白色 Batu Angin 通风花砖透气屏风墙。

2. 室内花砖光影中庭 (Interior Space)

阳光透过风砖在水磨石地面上投射几何阴影，设室内蕨类中庭。

🌟 风格 2 核心防热与美学优势解构

  • 双层墙防热屏风 (Double-Skin Facade)：Batu Angin 通风花砖隔绝正午热辐射，降低室内温度 3-5°C。
  • 自然穿堂风：利用风砖网格形成全天候微风流动，大幅减少空调依赖。
  • 度假村高级感：暖沙色粗抹灰结合热带蕨类景观，极富现代高端度假别墅气质。
STYLE PACKAGE 01

风格 1：居銮现代热带传统风格 (Modern Tropical Malay)

材质组合：赤陶红高坡屋顶 + 米白色波纹钢板外墙 + 暖木色 WPC 木塑格栅 + 柚木百叶通风塔

1. 外观正立面 (Front Exterior Elevation)

赤陶红斜屋顶与 50ft 度假村风入口，极简防热开窗。

2. 内部挑高大厅 (Interior Space)

裸露柚木屋架与中空采光窗，抛光混凝土地板与二层办公平台。

🔴 风格 1 设计解构

  • 高耸拔风塔 (Jack Roof)：中央 Raised Monitor 利用热浮力快速排出高空蓄热。
  • WPC 木塑格栅：暖木色格栅修饰开窗，兼顾防防热与马来传统 Kampung 建筑亲切感。
STYLE PACKAGE 03

风格 3：南洋殖民复古纯白谷仓 (Colonial Nanyang White Barn)

材质组合：全纯白波纹金属板墙面与屋顶 (最高太阳反射率 SRI) + 深色柚木百叶窗扇 (Plantation Shutters)

1. 外观正立面 (Front Exterior Elevation)

纯白谷仓轮廓搭配深色柚木百叶推拉门与高通风窗。

2. 纯白大厅与柚木地板 (Interior Space)

双倍层高纯白钢桁架 ceiling，柚木实木地板与百叶推拉门。

⚪ 风格 3 设计解构

  • 超高 SRI 太阳反射率：全纯白包覆极强反射太阳热辐射，室内吸热降至最低。
  • 南洋殖民风采：深色柚木百叶窗形成典雅对比，打造高雅奢华的私人会所质感。

💡 降本增效核心策略 (Cost Optimization Strategy)

  • ✅ 集装箱高强度模组化改造：利用二手 40ft HC 集装箱主体结构，切割开窗后补强 100mm 方钢，节约主体工期 40%。
  • ✅ 取消昂贵 50ft Clear Span：在中段增设两排 150mm×150mm H-beam 钢柱，直接节省 RM 35,000 巨型桁架费用。
  • ✅ 湿区背靠背共享管线：双厕所紧贴后墙，管路长度缩短 60%，防漏水风险降低。
  • ✅ 被动式防热拔风塔：靠自然热浮力排热，节约空调大功率工业电费达 45%。
