# Lot 7836 Kluang Modern Tropical Warehouse Bungalow Development Pack

> **项目名称**: 居銮 Lot 7836 50' × 75' 现代热带集装箱改建仓储别墅开发提案  
> **工程预算**: RM400,000 - RM450,000 Functional Baseline  
> **核心特征**: 50ft 面宽 × 75ft 进深 | 0-1 集装箱改造 | 拔风塔 (Jack Roof) 对流防热 | Batu Angin 通风花砖屏风墙  

---

## 🌟 项目亮点与包含内容

1. **马六甲风砖屏风墙 0-1 施工 GIF 动画**:
   * 从 40ft 原装集装箱定位、侧墙等离子切割与 100mm 方钢补强、模块拼接到 Batu Angin 花砖砌筑的全过程动态演示。
2. **独立风格包展示 (100% Grouped Style Packages)**:
   * **风格 01**: 居銮现代热带传统风格 (赤陶红屋顶 + 波纹钢板 + WPC 木塑格栅)
   * **风格 02**: 马六甲度假村风砖屏风墙 (主推 - 森林绿 PU 屋顶 + Batu Angin 双层透气屏风墙)
   * **风格 03**: 南洋殖民复古纯白谷仓 (纯白压型钢墙 + 深色柚木百叶窗)
3. **2D CAD 施工与工程节点图纸**:
   * **集装箱切割与拼接 Detail 图** (开口方钢补强、H-Beam 柱网焊接、风砖锚固)
   * **总平面规划图 (Site Plan)**
   * **正立面工程图 (Front Elevation)**
   * **2D 平面功能布局图 (Floor Plan)**
   * **MEP 水电与排水工程图 (MEP Plan)**

   ```bash
   git clone https://github.com/your-username/lot7836-development-pack.git
   cd lot7836-development-pack
   ```

   ```bash
   python3 -m http.server 8080
   ```
