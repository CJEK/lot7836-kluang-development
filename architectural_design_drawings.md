# Lot 7836 建筑设计控制图纸说明

> 本文件是 Lot 7836 的文字版母图控制说明。所有 CAD、渲染图、网页、成本估算和 AI 协作输出均须以本文件为准；如与旧图纸或旧文案冲突，以本文件和 `CODEX_GUIDE.md` 的硬性尺寸为准。

## 1. 设计基准与朝向

- 地块 / 建筑控制外框：**50ft（Jalan Pakis 正面面宽）× 75ft（向北进深）= 3,750 sqft**。
- 图纸下方为 Jalan Pakis；从道路向地块内看，75ft 方向向北延伸。
- 以下尺寸为方案控制尺寸，审批、结构、消防与施工图仍须由马来西亚注册专业人士复核。

## 2. 平面控制逻辑（不可改变）

| 区域 | 宽度 / 深度 | 控制要求 |
| --- | --- | --- |
| 西侧集装箱 | 8ft × 40ft | 40ft 长边沿 75ft 进深方向纵向摆放，位于前 40ft。 |
| 中央挑高中庭 | 34ft × 40ft（前段） | 保持无遮挡的主空间与穿堂风路径。 |
| 东侧集装箱 | 8ft × 40ft | 与西侧对称，40ft 长边同样沿进深方向纵向摆放。 |
| 后方钢构延伸段 | 50ft × 35ft | 前段 40ft 后连续延伸；设置湿区、仓储及阁楼平台。 |
| 总横向校核 | 8ft + 34ft + 8ft = 50ft | 任何立面、渲染或 CAD 不能将中央净跨改为 36ft 或横向摆放集装箱。 |

## 3. 屋顶与标高控制

- 屋檐标高：`+24'-0"`（约 7.3m）。
- 主屋顶：跨越 50ft 面宽的对称 30° 坡屋顶；半跨 = 25ft。
- 屋脊基座：`24ft + 25ft × tan(30°) = 38.43ft`，图纸标注采用 **`+38'-5"`**。
- Raised Jack Roof 拔风塔最高屋脊：**`+47'-0"`（约 14.33m）**。
- 所有屋顶、立面和展示文案须同时区分 `+38'-5"` 主屋脊基座与 `+47'-0"` 拔风塔顶，不能以旧版 `+36ft` 代替。

## 4. 阁楼与结构边界

- 900 sqft Mezzanine 仅位于后方 35ft 高顶轻钢结构区。
- 该区以 `+24'-0"` 屋檐控制；首层与阁楼各保留约 11ft（3.35m）净高。
- 严禁将 Mezzanine 放入 40ft HC 集装箱本体；其内部净高约 8'-10"（2.69m），不满足该空间逻辑。
- 集装箱开口、拆板或大跨度连接必须经结构工程师设计；本方案仅保留“切割后以 100mm × 100mm RHS 周边补强”的概念节点，不替代结构计算。

## 6. 热浮力烟囱效应自然排风预验算

### 输入与口径

| 项目 | 采用值 | 说明 |
| --- | ---: | --- |
| 室外干球温度 `To` | 32°C / 305.15K | 居銮炎热日间设计工况；不是全年气象设计年。 |
| 室内温度 `Ti` | 36°C / 309.15K | 给定内外温差 `ΔT=4°C`。烟囱效应只在室内更热时产生。 |
| 有效高度 `h` | 46ft / 14.021m | +1ft Batu Angin 进风标高至 +47ft Jack Roof 顶。 |
| 流量系数 `Cd` | 0.60 | 概念性锐边开口系数；须由实际百叶、风砖和防虫网压损复核。 |
| 进风 / 排风自由面积 | 各 3.0m² | 可施工的最低设计目标，指扣除风砖实体、网片和百叶后的净自由面积。 |
| 串联等效面积 `Aeff` | 2.121m² | `Aeff = (1/Ain² + 1/Aout²)^-0.5`；两个同为 3.0m² 的开口折算。 |

### 流量、CFM 与 ACH

采用 `Q = Cd × Aeff × √(2gh(Ti-To)/Ti)`，其中 `g=9.80665m/s²`：

```text
√(2 × 9.80665 × 14.021 × 4 / 309.15) = 1.886 m/s
Q = 0.60 × 2.121 × 1.886 = 2.401 m³/s
  = 5,087 CFM
```

建筑保守空气体积按完整 3,750 sqft 平面与 24ft 檐高、30° 屋顶三角附加体积计：

```text
V = 3,750 × [24 + (25 × tan(30°))/2] = 117,063 ft³
ACH = 5,087 × 60 / 117,063 = 2.61 次/小时
```

该公式可按有效开口面积线性缩放：在上述温差和高度下，每 **1.0m² Aeff** 约为 **2,398 CFM / 1.23 ACH**。实际风向、风速、室内门扇位置、网片堵塞和烟囱塔几何都会改变结果，故不得将此值作为保证风量。

### “被动式降温 3–5°C”的正确推演

此工况的可移除显热约为 `ρ × cp × Q × ΔT = 1.2 × 1,006 × 2.401 × 4 = 11.6kW`。因此，当 PU 隔热屋面、深檐、遮阳与风砖双层墙把日间剩余显热控制在约 11.6kW 或以下时，2.61 ACH 的自然排风可把无通风条件下的热积聚降低约 3–5°C（相对室内空气）。它**不能**使室温低于 32°C 室外干球温度；最终温度须以热负荷模拟、现场风速与实测数据确认。

## 7. 900 sqft Mezzanine 梁柱概念预验算

### 设计模型与截面假设

- 阁楼面积：900 sqft = 83.61m²，位置仅限后方 35ft 轻钢区。
- 楼面服务荷载：恒载 `Gk=1.5kN/m²` + 活荷载 `Qk=2.5kN/m²` = `4.0kN/m²`。
- 主梁：简支 250×125mm UB；为避免截面表版本差异，预验算采用不小于焊接 H250×125×6×9 的保守几何属性 `Ixx=38.93×10⁶mm⁴`、`Zx=311×10³mm³`、`E=200GPa`、`fy=275MPa`。采购型钢须以制造商证书复核实际 `Ixx/Zx`。
- 梁跨度 `L=17ft=5.182m`，每根梁的最大分担宽度限定为 `2.90m`，故服务线荷载 `q=4.0×2.90=11.60kN/m`。
- 柱：不小于 H150×150×7×10；采用 `A=3,910mm²`、弱轴 `r=37.94mm`、无侧向支撑高度 `3.35m`、有效长度系数 `K=1.0`。节点与整体侧向支撑尚未计入。

### 主梁结果

```text
服务弯矩 Mmax = qL²/8 = 11.60 × 5.182² / 8 = 38.93 kN·m
ULS 线荷载 qu = (1.35×1.5 + 1.5×2.5)×2.90 = 16.75 kN/m
ULS 弯矩 Mu = quL²/8 = 56.21 kN·m
概念抗弯强度 0.9×Zx×fy = 77.08 kN·m > 56.21 kN·m

服务挠度 fmax = 5qL⁴/(384EI) = 13.98mm
限值 L/360 = 5,182/360 = 14.39mm
结论：13.98mm < 14.39mm，通过；主梁分担宽度不得超过 2.90m。
```

### 柱结果与限制

一根梁在 ULS 下的端反力为 `quL/2=43.39kN`；一支中间柱若承托两根梁，概念轴力约为 `86.78kN`（未含柱自重、连接偏心、风震或屋顶荷载）。弱轴 Euler 临界力约 `990kN`；采用典型压杆折减后仍有大于 500kN 的量级容量，远高于该概念轴力。此结果只说明选型具有初步余量，不能豁免专业工程师对柱脚、梁柱连接、侧向稳定、楼板振动、火灾设计、荷载组合及基础的完整设计。
